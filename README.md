### 💡 Key Concepts

- JSX HTML jaisa dikhta hai but actually JavaScript hota hai  
- Functional component = simple JS function jo JSX return karta hai  
- Props = data passing between components  
- Reusability = ek component ko multiple baar use kar sakte hain  
- State change hone par UI re-render hota hai  
- useState se data control kar sakte hain  
- Event handling se interactivity aati hai
