# Episode 01 - Inception

## What is React
React is a JavaScript library used to build fast and interactive user interfaces.

## Why React
- Component based architecture
- Reusable components
- Fast rendering using Virtual DOM

## CDN vs npm
CDN: React can be used directly in HTML using a script tag.

Example:
<script src="https://unpkg.com/react@18/umd/react.development.js"></script>

npm: React can be installed in a project using npm and used in modern development setups.

## First React Element
React elements can be created using React.createElement().

Example:
const heading = React.createElement("h1", {}, "Hello World from React");

## Key Learning
React makes it easier to build complex UI using reusable components.
